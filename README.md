# WanderMind AI

Production-style travel planning application built with React + Vite.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel settings

- Framework: Vite
- Build command: npm run build
- Output directory: dist
- Install command: npm install

## Product areas

- Landing page
- Travel planning workspace
- Traveller profile selector
- Smart itinerary generation
- Hotel, flight and experience options
- Budget and checkout flow
- Promo code field
- Save, view, edit, cancel and delete trips
- Travel insights
- Responsive layout
